# desafio-uff-frontend

Frontend da Plataforma de Tarefas (To-Do List) - Desafio Fullstack UFF.

## Stack

- Vite + React 18 + TypeScript
- Tailwind CSS v4
- React Router, Axios, Zod, React Hook Form, Framer Motion

## Desenvolvimento

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
